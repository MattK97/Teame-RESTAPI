# Teame RestApi

This is a backend to the Teame application, it uses packages:

**Flask** 

**SQLAlchemy ORM** 

**Flask-Migrate**

It connects to PostgreSQL 13 database via psycopg2 (preferably) adapter

You can use this code to host the Teame application backend yourself, in which case you will need to create a Firebase project to 
- User authentication
- Notification
